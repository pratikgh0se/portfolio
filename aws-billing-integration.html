AWS Billing Data Integration

Company: Highspot

Dates: April 2024 - Present

Integrated AWS billing data to produce actionable insights on usage patterns, enabling cost-saving decisions.

Key Technologies & Tools

AWS (Cost and Usage Reports, Cost Explorer API, Lambda, Firehose, S3), Data Warehousing (e.g., Snowflake), New Relic, Data Analysis.

Challenge

Highspot incurred significant and growing cloud costs from its extensive use of AWS services. However, there was a critical lack of granular visibility and attribution regarding how specific services, individual teams, or distinct projects contributed to the overall expenditure. This opaque cost structure made it extremely difficult to identify inefficiencies, accurately attribute costs to business units, or make informed, data-driven decisions for effective cost optimization. Business leaders and engineering managers were often flying blind regarding their cloud spend.

Action

  • Meticulously identified and configured the most relevant AWS billing data sources, primarily setting up AWS Cost and Usage Reports (CUR) to be delivered regularly to a designated S3 bucket, and exploring/integrating with the AWS Cost Explorer API.
  • Designed and implemented a robust and scalable data pipeline to efficiently ingest the voluminous AWS billing data, leveraging AWS Lambda functions triggered by S3 events and AWS Kinesis Firehose.
  • Developed sophisticated processes to transform the raw, complex billing data into an actionable format, including parsing intricate CUR files and enriching data with internal metadata by applying strategic tagging strategies.
  • Created intuitive and comprehensive dashboards and reports, primarily leveraging tools like New Relic custom events/metrics and integrating with a dedicated business intelligence platform or data warehouse (e.g., Snowflake) for deeper financial analysis.
  • Performed ongoing, in-depth analysis of the integrated billing data, deriving actionable insights for cost-saving opportunities, such as rightsizing underutilized EC2 instances, optimizing storage tiers, and identifying idle or orphaned resources.
  • Presented these actionable recommendations to engineering teams, finance departments, and leadership, facilitating data-driven discussions and decisions on cloud cost optimization initiatives.

Result

  • The integration of AWS billing data provided unprecedented, granular visibility into AWS usage patterns and associated costs across Highspot.
  • This critical transparency enabled the organization to make informed, data-driven cost-saving decisions, directly leading to optimized cloud spending and improved resource efficiency across the entire AWS footprint.
  • The initiative significantly improved financial transparency and accountability for cloud resource consumption among various teams, fostering a culture of cost awareness and responsible cloud usage.

Review

This project provided critical transparency and enabled significant cost optimization for Highspot. My ability to design and implement robust data pipelines, perform in-depth data analysis, and translate complex financial insights into actionable recommendations demonstrated strong data engineering, analytical thinking, and business acumen. This initiative directly addressed a major business pain point related to cloud expenditure, showcasing my strategic value beyond technical execution.

AWS Cost Data ETL Pipeline (Python)

# Python implementation for AWS cost data processing
import boto3
import pandas as pd
from datetime import datetime, timedelta
from typing import Dict, List, Optional

class AWSCostAnalyzer:
    def __init__(self, aws_profile: str = 'default'):
        self.ce_client = boto3.client('ce', profile_name=aws_profile)
        self.s3_client = boto3.client('s3', profile_name=aws_profile)
        
    def extract_cost_and_usage_data(
        self, 
        start_date: str, 
        end_date: str,
        granularity: str = 'DAILY'
    ) -> Dict:
        """Extract cost and usage data from AWS Cost Explorer"""
        
        response = self.ce_client.get_cost_and_usage(
            TimePeriod={
                'Start': start_date,
                'End': end_date
            },
            Granularity=granularity,
            Metrics=['BlendedCost', 'UnblendedCost', 'UsageQuantity'],
            GroupBy=[
                {'Type': 'DIMENSION', 'Key': 'SERVICE'},
                {'Type': 'TAG', 'Key': 'Team'},
                {'Type': 'TAG', 'Key': 'Environment'}
            ]
        )
        
        return response
    
    def process_cost_data(self, raw_data: Dict) -> pd.DataFrame:
        """Transform and normalize cost data"""
        records = []
        
        for result in raw_data['ResultsByTime']:
            period_start = result['TimePeriod']['Start']
            
            for group in result['Groups']:
                # Extract dimensions and tags
                service = group['Keys'][0] if group['Keys'] else 'Unknown'
                team = group['Keys'][1] if len(group['Keys']) > 1 else 'Untagged'
                environment = group['Keys'][2] if len(group['Keys']) > 2 else 'Unknown'
                
                # Extract cost metrics
                blended_cost = float(group['Metrics']['BlendedCost']['Amount'])
                unblended_cost = float(group['Metrics']['UnblendedCost']['Amount'])
                
                records.append({
                    'date': period_start,
                    'service': service,
                    'team': team,
                    'environment': environment,
                    'blended_cost': blended_cost,
                    'unblended_cost': unblended_cost,
                    'currency': group['Metrics']['BlendedCost']['Unit']
                })
        
        df = pd.DataFrame(records)
        df['date'] = pd.to_datetime(df['date'])
        return df
    
    def detect_cost_anomalies(self, df: pd.DataFrame, threshold: float = 2.0) -> List[Dict]:
        """Detect cost anomalies using statistical analysis"""
        anomalies = []
        
        # Group by service and calculate rolling statistics
        for service in df['service'].unique():
            service_data = df[df['service'] == service].sort_values('date')
            service_data['rolling_mean'] = service_data['blended_cost'].rolling(7).mean()
            service_data['rolling_std'] = service_data['blended_cost'].rolling(7).std()
            
            # Detect anomalies using z-score
            service_data['z_score'] = (
                (service_data['blended_cost'] - service_data['rolling_mean']) / 
                service_data['rolling_std']
            )
            
            anomalous_points = service_data[
                abs(service_data['z_score']) > threshold
            ]
            
            for _, row in anomalous_points.iterrows():
                anomalies.append({
                    'date': row['date'],
                    'service': service,
                    'cost': row['blended_cost'],
                    'expected_cost': row['rolling_mean'],
                    'z_score': row['z_score'],
                    'severity': 'high' if abs(row['z_score']) > 3 else 'medium'
                })
        
        return anomalies

Automated Cost Reporting (AWS Lambda)

# Serverless cost reporting with AWS Lambda
import json
import boto3
from datetime import datetime, timedelta
import matplotlib.pyplot as plt
import seaborn as sns
from io import BytesIO

def lambda_handler(event, context):
    """Generate and distribute weekly cost reports"""
    
    # Initialize AWS clients
    ce_client = boto3.client('ce')
    ses_client = boto3.client('ses')
    s3_client = boto3.client('s3')
    
    # Calculate date range for weekly report
    end_date = datetime.now().date()
    start_date = end_date - timedelta(days=7)
    
    # Extract cost data
    cost_analyzer = AWSCostAnalyzer()
    raw_data = cost_analyzer.extract_cost_and_usage_data(
        start_date.isoformat(),
        end_date.isoformat()
    )
    
    df = cost_analyzer.process_cost_data(raw_data)
    
    # Generate cost summary
    total_cost = df['blended_cost'].sum()
    top_services = df.groupby('service')['blended_cost'].sum().sort_values(ascending=False).head(5)
    team_costs = df.groupby('team')['blended_cost'].sum().sort_values(ascending=False)
    
    # Detect anomalies
    anomalies = cost_analyzer.detect_cost_anomalies(df)
    
    # Generate visualization
    fig, ((ax1, ax2), (ax3, ax4)) = plt.subplots(2, 2, figsize=(15, 10))
    
    # Daily cost trend
    daily_costs = df.groupby('date')['blended_cost'].sum()
    ax1.plot(daily_costs.index, daily_costs.values)
    ax1.set_title('Daily Cost Trend')
    ax1.set_ylabel('Cost ($)')
    
    # Top services pie chart
    ax2.pie(top_services.values, labels=top_services.index, autopct='%1.1f%%')
    ax2.set_title('Top 5 Services by Cost')
    
    # Team cost comparison
    team_costs.plot(kind='bar', ax=ax3)
    ax3.set_title('Cost by Team')
    ax3.set_ylabel('Cost ($)')
    ax3.tick_params(axis='x', rotation=45)
    
    # Environment breakdown
    env_costs = df.groupby('environment')['blended_cost'].sum()
    ax4.bar(env_costs.index, env_costs.values)
    ax4.set_title('Cost by Environment')
    ax4.set_ylabel('Cost ($)')
    
    plt.tight_layout()
    
    # Save chart to S3
    img_buffer = BytesIO()
    plt.savefig(img_buffer, format='png', dpi=300, bbox_inches='tight')
    img_buffer.seek(0)
    
    s3_key = f"cost-reports/{end_date}/weekly-cost-analysis.png"
    s3_client.put_object(
        Bucket='highspot-cost-reports',
        Key=s3_key,
        Body=img_buffer.getvalue(),
        ContentType='image/png'
    )
    
    # Prepare email report
    report_html = generate_html_report(total_cost, top_services, team_costs, anomalies)
    
    # Send email report
    send_cost_report_email(ses_client, report_html, s3_key)
    
    return {
        'statusCode': 200,
        'body': json.dumps({
            'message': 'Cost report generated successfully',
            'total_cost': float(total_cost),
            'anomalies_detected': len(anomalies)
        })
    }
