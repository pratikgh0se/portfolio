Key Technologies & Tools

Situation

At EPAM Systems, many projects were experiencing inefficiencies and delays in their software delivery lifecycle. Application deployment and infrastructure provisioning were often manual, repetitive, and consequently, error-prone processes. This reliance on manual intervention led to inconsistent environments, prolonged release cycles, and a significant drain on developer and operations team time, hindering the ability to deliver software rapidly and reliably.

Task

My primary task as a Junior Software Engineer was to contribute to the modernization of the software delivery process. This involved designing and implementing end-to-end CI/CD pipelines, automating infrastructure provisioning using Python scripting and Terraform, and contributing to the development of reusable, scalable infrastructure management practices within a Microsoft Azure cloud environment.

Action

  • Actively participated in the design and implementation of end-to-end CI/CD pipelines, focusing on automation and consistency.
  • Utilized Docker to containerize various applications, writing Dockerfiles and optimizing image sizes for consistent execution across environments.
  • Gained hands-on experience in configuring Jenkins pipelines to automate the entire CI/CD workflow, including automated builds, unit/integration tests, and orchestrating deployments to Azure environments.
  • Integrated Jenkins with Microsoft Azure DevOps and directly with Azure services to facilitate automated deployments to Azure App Services, VMs, and contributing to early deployments on Azure Kubernetes Service (AKS).
  • Developed a suite of Python scripts for infrastructure automation tasks, leveraging Azure SDKs to programmatically provision, configure, and manage Azure resources.
  • Actively contributed to the development of Terraform-based Azure Proof of Concept (POC) libraries, writing HCL code and creating reusable Terraform modules for common infrastructure patterns.
  • Participated in internal "Azure micro-communities" to share best practices and collectively develop robust infrastructure configurations.
  • Engaged proactively with development teams, Quality Assurance (QA), and other operations personnel, understanding their deployment needs and integrating automation solutions seamlessly.
  • Contributed to providing initial training and support to ensure smooth adoption of new CI/CD processes and automated tooling.

Result

  • The implementation of automated CI/CD pipelines streamlined application deployments, leading to reduced build times and faster, more reliable releases.
  • Python-based automation tools drastically cut down manual infrastructure provisioning efforts, improving overall deployment efficiency and consistency.
  • My contributions to Terraform POC libraries laid the foundational groundwork for more scalable, modular, and version-controlled infrastructure management practices within EPAM Systems.
  • Overall, these efforts significantly enhanced EPAM Systems' operational efficiency and agility in software delivery.

Review

The hands-on experience with key technologies like Docker, Jenkins, Microsoft Azure, Python automation, and Terraform was invaluable for my early career development. Jenkins Pipeline Configuration

// Jenkins pipeline for automated application deployment
pipeline {
    agent any
    
    environment {
        AZURE_CREDENTIALS = credentials('azure-service-principal')
        DOCKER_REGISTRY = 'epamregistry.azurecr.io'
        APPLICATION_NAME = 'webapp'
    }
    
    parameters {
        choice(name: 'ENVIRONMENT', choices: ['dev', 'staging', 'prod'], 
               description: 'Target deployment environment')
        string(name: 'VERSION', defaultValue: 'latest', 
               description: 'Application version tag')
    }
    
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/epam/webapp.git'
            }
        }
        
        stage('Build Application') {
            steps {
                sh 'mvn clean package'
                archiveArtifacts artifacts: 'target/*.jar', fingerprint: true
            }
        }
        
        stage('Unit Tests') {
            steps {
                sh 'mvn test'
                publishTestResults testResultsPattern: 'target/surefire-reports/*.xml'
            }
        }
        
        stage('Build Docker Image') {
            steps {
                script {
                    def image = docker.build("${APPLICATION_NAME}:${params.VERSION}")
                    docker.withRegistry("https://${DOCKER_REGISTRY}", 'azure-registry-credentials') {
                        image.push()
                        image.push('latest')
                    }
                }
            }
        }
        
        stage('Deploy Infrastructure') {
            steps {
                sh '''
                    python3 infrastructure/deploy.py \
                        --environment ${ENVIRONMENT} \
                        --application ${APPLICATION_NAME} \
                        --version ${VERSION}
                '''
            }
        }
        
        stage('Deploy Application') {
            steps {
                sh '''
                    az webapp config container set \
                        --name ${APPLICATION_NAME}-${ENVIRONMENT} \
                        --resource-group ${APPLICATION_NAME}-rg \
                        --docker-custom-image-name ${DOCKER_REGISTRY}/${APPLICATION_NAME}:${VERSION} \
                        --docker-registry-server-url https://${DOCKER_REGISTRY}
                '''
            }
        }
        
        stage('Health Check') {
            steps {
                script {
                    def healthUrl = "https://${APPLICATION_NAME}-${params.ENVIRONMENT}.azurewebsites.net/health"
                    timeout(time: 5, unit: 'MINUTES') {
                        waitUntil {
                            script {
                                def response = sh(
                                    script: "curl -s -o /dev/null -w '%{http_code}' ${healthUrl}",
                                    returnStdout: true
                                ).trim()
                                return response == '200'
                            }
                        }
                    }
                }
            }
        }
    }
    
    post {
        always {
            cleanWs()
        }
        success {
            emailext(
                subject: "‚úÖ Deployment Success: ${APPLICATION_NAME} v${params.VERSION}",
                body: "Application deployed successfully to ${params.ENVIRONMENT}",
                to: "${env.CHANGE_AUTHOR_EMAIL}"
            )
        }
        failure {
            emailext(
                subject: "‚ùå Deployment Failed: ${APPLICATION_NAME} v${params.VERSION}",
                body: "Deployment failed. Please check the build logs.",
                to: "${env.CHANGE_AUTHOR_EMAIL}"
            )
        }
    }
}

Python Infrastructure Automation Script

# Python script for automated Azure infrastructure provisioning
import argparse
import json
import subprocess
import sys
from typing import Dict, List

class AzureInfrastructureDeployer:
    def __init__(self, environment: str, application_name: str):
        self.environment = environment
        self.application_name = application_name
        self.resource_group = f"{application_name}-{environment}-rg"
        
    def create_resource_group(self) -> bool:
        """Create Azure resource group if it doesn't exist"""
        try:
            cmd = [
                "az", "group", "create",
                "--name", self.resource_group,
                "--location", "eastus",
                "--tags", f"Environment={self.environment}",
                f"Application={self.application_name}"
            ]
            
            result = subprocess.run(cmd, capture_output=True, text=True)
            if result.returncode == 0:
                print(f"‚úì Resource group {self.resource_group} created/verified")
                return True
            else:
                print(f"‚úó Failed to create resource group: {result.stderr}")
                return False
                
        except Exception as e:
            print(f"‚úó Error creating resource group: {e}")
            return False
    
    def deploy_app_service_plan(self) -> bool:
        """Deploy Azure App Service Plan"""
        try:
            plan_name = f"{self.application_name}-{self.environment}-plan"
            
            # Check if plan exists
            check_cmd = [
                "az", "appservice", "plan", "show",
                "--name", plan_name,
                "--resource-group", self.resource_group
            ]
            
            result = subprocess.run(check_cmd, capture_output=True, text=True)
            if result.returncode == 0:
                print(f"‚úì App Service Plan {plan_name} already exists")
                return True
            
            # Create new plan
            create_cmd = [
                "az", "appservice", "plan", "create",
                "--name", plan_name,
                "--resource-group", self.resource_group,
                "--sku", "S1" if self.environment == "prod" else "B1",
                "--is-linux"
            ]
            
            result = subprocess.run(create_cmd, capture_output=True, text=True)
            if result.returncode == 0:
                print(f"‚úì App Service Plan {plan_name} created successfully")
                return True
            else:
                print(f"‚úó Failed to create App Service Plan: {result.stderr}")
                return False
                
        except Exception as e:
            print(f"‚úó Error deploying App Service Plan: {e}")
            return False
    
    def deploy_web_app(self, version: str) -> bool:
        """Deploy Azure Web App"""
        try:
            app_name = f"{self.application_name}-{self.environment}"
            plan_name = f"{self.application_name}-{self.environment}-plan"
            
            # Check if app exists
            check_cmd = [
                "az", "webapp", "show",
                "--name", app_name,
                "--resource-group", self.resource_group
            ]
            
            result = subprocess.run(check_cmd, capture_output=True, text=True)
            if result.returncode == 0:
                print(f"‚úì Web App {app_name} already exists")
                return True
            
            # Create new web app
            create_cmd = [
                "az", "webapp", "create",
                "--name", app_name,
                "--resource-group", self.resource_group,
                "--plan", plan_name,
                "--deployment-container-image-name", f"nginx:latest"
            ]
            
            result = subprocess.run(create_cmd, capture_output=True, text=True)
            if result.returncode == 0:
                print(f"‚úì Web App {app_name} created successfully")
                
                # Configure app settings
                self.configure_app_settings(app_name)
                return True
            else:
                print(f"‚úó Failed to create Web App: {result.stderr}")
                return False
                
        except Exception as e:
            print(f"‚úó Error deploying Web App: {e}")
            return False
    
    def configure_app_settings(self, app_name: str) -> bool:
        """Configure application settings"""
        try:
            settings = {
                "ENVIRONMENT": self.environment,
                "APPLICATION_NAME": self.application_name,
                "LOG_LEVEL": "INFO" if self.environment == "prod" else "DEBUG"
            }
            
            for key, value in settings.items():
                cmd = [
                    "az", "webapp", "config", "appsettings", "set",
                    "--name", app_name,
                    "--resource-group", self.resource_group,
                    "--settings", f"{key}={value}"
                ]
                
                subprocess.run(cmd, capture_output=True, text=True)
            
            print(f"‚úì App settings configured for {app_name}")
            return True
            
        except Exception as e:
            print(f"‚úó Error configuring app settings: {e}")
            return False
    
    def deploy_infrastructure(self, version: str) -> bool:
        """Deploy complete infrastructure stack"""
        print(f"üöÄ Starting infrastructure deployment for {self.application_name} v{version}")
        
        steps = [
            ("Creating resource group", self.create_resource_group),
            ("Deploying App Service Plan", self.deploy_app_service_plan),
            ("Deploying Web App", lambda: self.deploy_web_app(version))
        ]
        
        for step_name, step_function in steps:
            print(f"\nüì¶ {step_name}...")
            if not step_function():
                print(f"‚ùå Infrastructure deployment failed at: {step_name}")
                return False
        
        print(f"\nüéâ Infrastructure deployment completed successfully!")
        return True

def main():
    parser = argparse.ArgumentParser(description='Deploy Azure infrastructure')
    parser.add_argument('--environment', required=True, choices=['dev', 'staging', 'prod'])
    parser.add_argument('--application', required=True, help='Application name')
    parser.add_argument('--version', required=True, help='Application version')
    
    args = parser.parse_args()
    
    deployer = AzureInfrastructureDeployer(args.environment, args.application)
    success = deployer.deploy_infrastructure(args.version)
    
    sys.exit(0 if success else 1)

if __name__ == "__main__":
    main()

# Terraform configuration for Azure infrastructure (POC)
terraform {
  required_providers {
    azurerm = {
      source  = "hashicorp/azurerm"
      version = "~> 3.0"
    }
  }
}

provider "azurerm" {
  features {}
}

variable "environment" {
  description = "Environment name"
  type        = string
  validation {
    condition = contains(["dev", "staging", "prod"], var.environment)
    error_message = "Environment must be dev, staging, or prod."
  }
}

variable "application_name" {
  description = "Application name"
  type        = string
}

variable "location" {
  description = "Azure region"
  type        = string
  default     = "East US"
}

# Resource Group
resource "azurerm_resource_group" "main" {
  name     = "${var.application_name}-${var.environment}-rg"
  location = var.location

  tags = {
    Environment = var.environment
    Application = var.application_name
    ManagedBy   = "Terraform"
  }
}

# App Service Plan
resource "azurerm_service_plan" "main" {
  name                = "${var.application_name}-${var.environment}-plan"
  resource_group_name = azurerm_resource_group.main.name
  location           = azurerm_resource_group.main.location
  os_type            = "Linux"
  sku_name           = var.environment == "prod" ? "S1" : "B1"

  tags = {
    Environment = var.environment
    Application = var.application_name
  }
}

# Web App
resource "azurerm_linux_web_app" "main" {
  name                = "${var.application_name}-${var.environment}"
  resource_group_name = azurerm_resource_group.main.name
  location           = azurerm_service_plan.main.location
  service_plan_id    = azurerm_service_plan.main.id

  site_config {
    application_stack {
      docker_image     = "nginx"
      docker_image_tag = "latest"
    }
  }

  app_settings = {
    "ENVIRONMENT"      = var.environment
    "APPLICATION_NAME" = var.application_name
    "LOG_LEVEL"       = var.environment == "prod" ? "INFO" : "DEBUG"
  }

  tags = {
    Environment = var.environment
    Application = var.application_name
  }
}

# Container Registry
resource "azurerm_container_registry" "main" {
  count               = var.environment == "prod" ? 1 : 0
  name                = "${replace(var.application_name, "-", "")}registry"
  resource_group_name = azurerm_resource_group.main.name
  location           = azurerm_resource_group.main.location
  sku                = "Basic"
  admin_enabled      = true

  tags = {
    Environment = var.environment
    Application = var.application_name
  }
}

# Outputs
output "web_app_url" {
  value = "https://${azurerm_linux_web_app.main.default_hostname}"
}

output "resource_group_name" {
  value = azurerm_resource_group.main.name
